# Homebrew Tap for Forge

Official Homebrew formulae for [Forge](https://github.com/humancto/forge-lang), the internet-native programming language.

## Install

```bash
brew install humancto/tap/forge
```

Or:

```bash
brew tap humancto/tap
brew install forge
```

## Upgrade

```bash
brew upgrade forge
```

## Uninstall

```bash
brew uninstall forge
```
